

[지정변수]	
a1 ~ a6 까지의 지정변수가 존재한다. 지정변수의 값은 위치정보로 할당한다.	
구분자가 있는 테이블의 경우 인덱스, 구분자가 없는경우 시작위치, 개수를 입력한다	
ex1) Line: abcdefg	위치: 1,2 -> bc
ex2) Line: aa;bb;cc;dd	위치: 3 -> dd

[함수]
SetContains 

	
[기능]	
Split	
- 파일을 SplitBy 기준에 의해 분할한다.	
ex1) a1	
ex2) sub(0,3)	
ex3) Dict1(a1)

Group
- 여러 파일을 Key로 그룹화 하여 SplitBy 조건으로 분할 한 후 WriteLine 기준으로 라인을 생성한다.
텍스트 파일이 그룹화 될 경우 Sum, Count, First, Last 함수의 사용이 가능 해 진다.
Key의 Loop는 0번 파일을 기준으로 한다. 1~n번 파일의 경우 Key탐색 대상이 된다.
만약 탐색이 되지 않을 경우 Sum, Count 값은 0, First, Last 값은 "Not Found"가 된다.

ex1) 0번 파일의 a1부분의 합계를 계산 Sum(0, a1) 
ex2) 1번 파일의 Key에 해당하는 라인 수 Count(1)
ex3) 3번 파일의 Key에 해당하는 라인의 첫번째 값 First(1)
	
Filter	
- 파일을 FilterBy 조건에 일치하는 라인을 걸러낸다.	
ex1) a1 = "LA001"	
ex2) ToInt(a1) = 0
ex3) Double(a1) < 5.0
ex4) SetContains(a1)
	
Sort	
- 파일을 SortBy 기준으로 정렬한다. SortBy1은 필수입력	
ex1) SortBy1: a1 , SortBy2: Int(a2)	
	
Select	
- 라인의 일부분을 선택하여 파일을 재구성한다.	
ex1) Join("|", a1, a2, a3, a4)
ex2) Join(";", sub(0,2), sub(2,4), sub(4,10), sub(10,15), ...)
	
Distinct	
- 파일을 DistinctBy 기준으로 중복제거 한다.	
ex1) a1 + a2
	
[함수]	
함수명	설명
Int(string a1)	문자열을 정수 값으로 변환한다.
Double(string a1)	문자열을 실수 값으로 변환한다.
SetContains(string a1)	Set에 a1이 포함되어 있으면 True
DictContainsKey(string a1)	a1이 Dict의 Key값에 포함되어 있으면 True
Dict(string a1)	Dict의 Key(a1)에 해당하는 Value값 반환, Key가 없을 경우 "NotMatch" 반환
Dict(string a1, string notMatchStr)	Dict의 Key(a1)에 해당하는 Value값 반환, Key가 없을 경우 notMatchStr 반환
Sub(int idx)	구분자 구조의 라인에서 인덱스 n1 위치의 문자열 값 반환
Sub(int start, int length)	고정폭 구조의 라인에서 시작위치(start)에서 length개 만큼의 문자열 값 반환
Join(string seperator, string a1, string a2, ...)	a1,a2...을 seperator로 연결
